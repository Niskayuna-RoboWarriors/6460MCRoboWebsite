How to set up Fabric for Minecraft Java

To run mods, it is highly recommended to use Fabric. Download fabric at https://fabricmc.net, download and run that .exe file off of Fabric,
and make sure you install it in your .minecraft folder. After you installed Fabric, head over to your .minecraft folder (C:\Users\*account name*\AppData\Roaming\.minecraft), and create a folder called >mods<. (It is case sensitive).
Once you're done installing and adding the >mods< folder, you can start installing mods to that folder you want to run!
NOTE: MAKE SURE YOU INSTALL THE FABRIC API OFF OF THE FABRIC WEBSITE, AND THROW IT IN YOUR MODS FOLDER. (Any mods you install will not work unless if you have the fabric API).
If you cannot find the Fabric API off of the website, click here to download it
